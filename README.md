# Java Beginner Projects

This repository contains beginner-level Java console projects and programming exercises.

The projects were developed to practice basic Java syntax, variables, user input, conditional statements and simple calculations.

## Projects

### 1. Taximeter Program

A simple Java console application that calculates taxi fare based on distance.

### 2. Hypotenuse Calculator

A basic Java program that calculates the hypotenuse of a right triangle using side lengths.

---

## Topics Covered

- Java syntax
- Variables
- User input with Scanner
- Conditional statements
- Basic arithmetic operations
- Console-based programming

---

## Technologies Used

- Java
- IntelliJ IDEA
